# Bokeh Presentation - Christchurch Python Meetup

Code samples and notes from Bokeh Presentation to the Christchurch Python Meetup.

All code written in Python 3.

## Installation

Clone (or Download above) using:

```bash
git clone https://github.com/andrewvillazon/chch-python-bokeh.git
```

_* Optional, but recommoned using a virtual environment with this step._

Then use the package manager [pip](https://pip.pypa.io/en/stable/) to install the required libraries from the requirements file.

```bash
pip install -r requirements.txt
```
